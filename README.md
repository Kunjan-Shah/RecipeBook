# RecipeBook

## Commands
Clone the repository

To run frontend on ```localhost:3000``` run the following commands:
```
cd frontend
npm i
npm start
```

To run backend on ```localhost:4000``` run the following commands:
```
cd backend
npm i
node server.js
```
